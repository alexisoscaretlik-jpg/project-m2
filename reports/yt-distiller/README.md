# reports/yt-distiller

Owner: `yt-distiller`. Daily log of the `cron/watch-kevin` pipeline plus weekly quality samples.

Filename: `YYYY-MM-DD.md` for daily; `samples/YYYY-WW.md` for weekly QA samples.

Each daily entry: cron `results` array verbatim (id, status, title, error?). Each weekly sample: 1 random distilled note re-checked against source video, scored on signal density, ticker accuracy, paraphrase strength, language match.
