## Master Algorithm Summary
To write like Snowball:
1. **Hook:** Start personal or provocative.
2. **Simplify:** No jargon without a definition.
3. **Visualize:** Use numbers to show the 'cost of doing nothing'.
4. **Interact:** Use 'tu' and speak to the reader's specific goals.
5. **Optimize:** Always include one 'Pro-tip' that feels like an insider secret.
